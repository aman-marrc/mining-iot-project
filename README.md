# 🚀 Mining Vehicle IoT Tracking System

## 📌 Project Overview
This project is a GPS-based IoT system that detects:
- Route deviation
- Idle behavior
- Simulated fuel consumption

## ⚙️ Tech Stack
- Python (Flask)
- Mapbox (Frontend Map)
- JavaScript
- Render (Backend Deployment)
- Netlify (Frontend Deployment)

## 🧠 How It Works
Simulator → Backend API → Map Dashboard

## 🚗 Features
- Live vehicle tracking
- Route visualization
- Idle detection alerts
- Route deviation alerts
